# BrewFinder  
A small React project for exploring breweries across the US.  
Built as part of a school assignment to practice React, routing, API fetching, and responsive CSS.

---

## Project Structure (simplified)

```
src/
  api/
    breweries.js
  components/
    Header/
    Navigation/
    BreweryCard/
    BreweryList/
  pages/
    Home/
    About/
    Breweries/
    Contact/
    Details/
  assets/
  App.jsx
  main.jsx
```

---

## API Used
This project uses the **Open Brewery DB** public API:

```
https://api.openbrewerydb.org/v1/breweries
```

No API key required.

---

## Responsive Design
The layout is built **mobile‑first**, then enhanced for larger screens.  
The header and footer stay fixed, and cards are centered using CSS Grid.

Each page has its own background image for a more “app‑like” feel.

---

## Notes
This project was created for learning purposes.  
The goal was to focus on:
- clean component structure  
- readable code  
- simple API handling  
- styling without frameworks  
- making the UI feel cohesive and friendly
